# herdle

> **Wrangle the herd, spot the hurdles.**

A self-contained CLI for tracking in-progress work across many git/GitHub
projects at a glance — built around [tk](https://github.com/wedow/ticket) and
[superpowers](https://github.com/obra/superpowers).

🚧 **Early development.** Not yet usable — design and scaffolding in progress.
